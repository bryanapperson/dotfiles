# Awesome WM

Compositor: `tryone-comton`
Terminal: `alacritty`

## Colors

Colors are set in `dotfiles/.config/theme.lua`, however they can be overridden by the colors in `~/.Xresources`, which is also the place to set DPI for high resolution scaling.

## Dependencies

```
tryone-compton
alacritty
maim
nm_applet
paprefs
pasystray
blueman
```
